# Sourav V Salunke - Personal Portfolio

Welcome to the personal portfolio of Sourav V Salunke, a Full Stack Developer specializing in the MERN stack with a passion for designing interactive user interfaces and intelligent scalable architectures.

## ✨ Features
- **Cinematic Space Theme**: Beautifully animated CSS space background featuring glowing elements, shooting stars, and a milky way backdrop.
- **GSAP Animations**: Smooth scrolling and advanced scroll-triggered visual effects.
- **EmailJS Integration**: Fully functioning contact suite with smart automated replies.
- **Tailwind CSS**: Pixel-perfect aesthetic and fully responsive mobile design.

## 🚀 Running Locally
To run this application locally on your machine:

1. Ensure you have Node.js installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:3000` in your browser.

## 🎵 Customizing Audio
If you wish to change the background music again, simply drop any `.mp3` file into the `public/sounds/` directory and rename it to `song.mp3`!
